## Eficiencia como meio

Eficiencia tem valor quando protege energia para aquilo que nao pode ser automatizado.

## Perguntas antes de otimizar

- Isso melhora a experiencia humana?
- Isso reduz desgaste recorrente?
- Isso preserva espaco para pensar?

```js
function shouldOptimize({ humanImpact, recurringPain }) {
  return humanImpact && recurringPain;
}
```
