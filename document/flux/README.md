# Fluxogramas e Diagramas (Mermaid)

Esta pasta centraliza a documentacao visual do projeto em Mermaid, com base nos documentos tecnicos ja existentes em `document/docs/**` e `document/modules/**`.

## Objetivo

- Dar uma visao rapida da arquitetura, navegacao e fluxos principais.
- Manter diagramas versionados junto ao codigo.
- Evitar divergencia entre implementacao e documentacao.

## Como visualizar

- GitHub: abra qualquer arquivo `.md` desta pasta para ver o Mermaid renderizado automaticamente.
- Local: use a visualizacao de Markdown da sua IDE (ex.: VS Code) com suporte a Mermaid.

## Como manter

- Atualize primeiro os documentos-fonte tecnicos.
- Ajuste os diagramas correspondentes nesta pasta.
- Sempre inclua a secao **Fonte** no topo de cada diagrama, apontando os arquivos usados.
- Siga o padrao descrito em `99-convencoes-mermaid.md`.
